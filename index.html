～ショートカット一覧～

必須級

  • コピー
  • ペースト
  • 再読み込み
  • 次に進む
  • 前に戻る
  • タブの新規作成
  • タブの削除
  • 画面の分割

使えたら便利

  • タブの復元
  • ウィンドウの新規作成
  • ウィンドウの削除
  • 仮想デスクトップ
  • コピー履歴表示
  • ページ内検索
  • スクリーンショット
  • スニッピング
  • 画面の録画
  • ウィンドウをすべて最小化
  • アイコンの表示形式を変更
  • ファイルの名前を変更

たまに使う

  • 右クリック
  • ゲームバーを開く
  • タスクバーのアプリを開く
  • タスクマネージャーを開く

必須級

コピー Ctrl + C
ペースト Ctrl + V
再読み込み Ctrl + R
次に進む Alt + →
前に戻る Alt + ←
タブの新規作成 Ctrl + T
タブの削除 Ctrl + W
画面の分割 Windows + → ←

使えたら便利

タブ ウィンドウ デスクトップ
新規作成 Ctrl + T Ctrl + N Windows + Ctrl + D
削除 Ctrl + W Alt + F4 Windows + Ctrl + F4
切り替え Ctrl + Tab Alt + Tab Windows + Ctrl + Tab
Windows + Ctrl + → ←
復元 Ctrl + Shift + T Ctrl + Shift + T ✕
コピー履歴表示 Windows + V
ページ内検索 Ctrl + F
スクリーンショット Windows + Alt + Print Screen
スニッピング Windows + Shift + S
画面の録画 Windows + Alt + R
ウィンドウをすべて最小化 Windows + D
アイコンの表示形式を変更 Ctrl + Shift + 1 ～ 8
ファイルやフォルダーの名前を変更 F2

たまに使う

右クリック Shift + F10
タスクマネージャーを開く Ctrl + Shift + Esc
ゲームバーを開く Windows + G
タスクバーのアプリを開く Windows + 1 ～ 8
